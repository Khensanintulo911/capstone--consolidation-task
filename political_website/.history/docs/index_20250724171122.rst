Political Candidate Website Documentation
=========================================

Welcome to the Political Candidate Website documentation. This Django-based web application 
provides a platform for political candidates to showcase their profiles, engage with voters,
and manage their online presence.

Features
--------

* Candidate profile management
* User authentication and registration
* Contact form for voter feedback
* Administrative interface
* Responsive design
